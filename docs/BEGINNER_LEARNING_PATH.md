# OriLang Beginner Learning Path

This guide provides a structured learning path for beginners, with 10 ordered samples that progressively introduce key concepts.

## Learning Path Overview

| # | Sample | Learning Goal | Key Concepts |
|---|--------|---------------|--------------|
| 1 | `beginner_01_hello.ori` | Basic output | `say` statement, string literals |
| 2 | `beginner_02_variables.ori` | Variable declaration | `hold` keyword, variable assignment |
| 3 | `beginner_03_loops.ori` | Loop constructs | `loop` keyword, loop conditions |
| 4 | `beginner_04_conditionals.ori` | Conditional logic | `when` / `else when` / `else` |
| 5 | `beginner_05_functions.ori` | Function definition | `fold` keyword, function calls |
| 6 | `beginner_06_strings.ori` | String operations | String concatenation, `len` |
| 7 | `beginner_07_arrays.ori` | Array basics | Array declaration, indexing |
| 8 | `beginner_08_when_chain.ori` | Chained conditionals | Multiple `when` statements |
| 9 | `beginner_09_math_chain.ori` | Math operations | Arithmetic operators, expressions |
| 10 | `beginner_10_bool_logic.ori` | Boolean logic | `yes`/`no`, logical operators |

## Detailed Learning Goals

### 1. Basic Output (`beginner_01_hello.ori`)
- Understand how to display text using `say`
- Learn about string literals and basic syntax

### 2. Variables (`beginner_02_variables.ori`)
- Learn to declare variables with `hold`
- Understand variable assignment and naming conventions

### 3. Loops (`beginner_03_loops.ori`)
- Master the `loop` construct for repeated execution
- Learn loop conditions and termination

### 4. Conditionals (`beginner_04_conditionals.ori`)
- Use `when` / `else when` / `else` for decision making
- Understand comparison operators (`==`, `!=`, `<`, `>`, `<=`, `>=`)

### 5. Functions (`beginner_05_functions.ori`)
- Define functions with `fold`
- Understand function parameters and return values
- Learn the `give` keyword for returning values

### 6. Strings (`beginner_06_strings.ori`)
- Work with string data types
- Practice string concatenation using `+`
- Use built-in functions like `len`

### 7. Arrays (`beginner_07_arrays.ori`)
- Create and manipulate arrays
- Access elements by index
- Use `push` to add elements

### 8. Chained Conditionals (`beginner_08_when_chain.ori`)
- Implement complex decision logic
- Chain multiple `when` statements
- Handle multiple conditions

### 9. Math Operations (`beginner_09_math_chain.ori`)
- Perform arithmetic operations
- Understand operator precedence
- Work with numeric expressions

### 10. Boolean Logic (`beginner_10_bool_logic.ori`)
- Understand boolean values (`yes`/`no`)
- Use logical operators (`&&`, `||`, `!`)
- Combine boolean expressions

## How to Use This Path

1. **Start with sample #1** and work your way through in order
2. **Read the code** in each sample before running it
3. **Predict the output** before executing the sample
4. **Run the sample** using `ori run samples/<sample_name>.ori`
5. **Compare your prediction** with the actual output
6. **Experiment** by modifying the code and observing changes
