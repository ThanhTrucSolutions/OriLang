# Common Ori Beginner Pitfalls

When starting with OriLang, watch out for these common mistakes:

1. **Missing Semicolons**: Unlike Python, Ori requires semicolons (`;`) at the end of statements.
2. **Variable Declaration**: Always use `let` to declare a new variable.
3. **Function Syntax**: Use the `fn` keyword, not `function` or `def`.
4. **String Concatenation**: Use the `+` operator to combine strings.
