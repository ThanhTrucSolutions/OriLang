# Beginner Learning Path

This guide walks through the OriLang beginner samples in recommended order.

## Phase 1: Basics (samples 01-06)
1. **hello** — Your first Ori program
2. **variables** — Storing values with `hold`
3. **loops** — Repeating with `when` blocks
4. **conditionals** — Branching with `when`
5. **functions** — Defining and calling functions
6. **strings** — String operations and concatenation

## Phase 2: Data Structures (samples 07-10)
7. **arrays** — Working with lists
8. **when_chain** — Multi-branch logic
9. **math_chain** — Chained arithmetic
10. **bool_logic** — Boolean operations

## Phase 3: Patterns (samples 11-20)
11. **countdown** — Loop decrement pattern
12. **max_of_two** — Comparison pattern
13. **double_loop** — Nested loops
14. **sum_range** — Accumulator pattern
15. **echo_twice** — Repetition pattern
16-20. More patterns...

## Phase 4: Wave 2 (samples 27-33)
27. **countdown** — Advanced countdown with messages
28. **max_of_three** — Three-way comparison
29. **echo_name** — String length and echo
30. **sum_1_to_n** — Summation loop
31. **even_odd** — Modulo and branching
32. **greeting_times** — Repeated greeting
33. **min_max** — Finding extremes

## Running Samples
```bash
ori run samples/beginner_01_hello.ori
```
